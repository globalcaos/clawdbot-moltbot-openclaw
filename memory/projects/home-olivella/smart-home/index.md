## Design Philosophy

Oscar prefers:
- **Wired connections** for reliability
- **Local control** over cloud dependency
- **Open protocols** where possible
- **Practical automation** over gimmicks

- Access points

## Integration Goals

Connect with:
- Solar monitoring (energy system)
- Security (future)
- Climate control
- Irrigation (garden)

## Avoid

- Battery-powered sensors (unreliable, maintenance burden)
- Cloud-only devices (privacy, reliability concerns)
- Proprietary ecosystems (lock-in)
